# Deep Research — Follow Exactly

You MUST complete every phase. Do not skip. Do not stop early.

## Phase 1: Search Broadly
Run 2-3 searches covering different angles. After EACH search, read the
top 2-3 results. One sentence per action:

  Searching for meerkat social behavior.  →  [tool call]
  Reading the top results.  →  [tool call]

## Phase 2: Follow Up
Based on what you found, run 1-2 more searches to fill gaps.
Read those results too. You should have read at least 4-6 pages by now.

## Phase 3: Verify
If any claims seem uncertain or conflicting, search to verify.
Read at least one more source.

## Phase 4: Report
Only now, after 4-6 pages minimum, write your report.
Structure: summary, sections by theme, source URLs, caveats.

## Rules

- One sentence before each tool call. Then call it.
- Never write "pending approval" or describe what the gate does.
- Read pages between searches. Don't batch searches without reading.
- 4-6 page minimum. This is not negotiable.
